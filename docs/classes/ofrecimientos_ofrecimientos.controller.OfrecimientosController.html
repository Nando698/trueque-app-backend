    • aceptar(
          id: string,
          req: AuthRequest,
      ): Promise<
          { contacto: { correo: string; nombre: string }; mensaje: string },
      >

      Acepta un ofrecimiento pendiente de la lista de recibidos. Al aceptar:

      • Cambia el estado del ofrecimiento a ACEPTADO
      • Marca la oferta como FINALIZADA
      • Rechaza automáticamente los demás ofrecimientos

      Parameters

      • id: string

        ID del ofrecimiento a aceptar

      • req: AuthRequest

        Request autenticada, contiene user.id del propietario de la oferta

      Returns Promise<{ contacto: { correo: string; nombre: string }; mensaje: string }>

      Objeto con mensaje de éxito y datos de contacto del oferente
